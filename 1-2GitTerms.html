1-2 Git Terms

Version Control Software: Is a backup. They are a database that keeps track of the changes made to files.

Add: Is a command that informs git that there is a change in a file that the user wants to git to get ready to commit.

Commit: Is a command that is used as a save. It shows all variations in a file timeline.

Push: Is a command that uploads the committed file from the local repository to the remote repository.

Pull: Is a command that takes and downloads changes made in the remote repository and updates the local repository with the changes.

Clone: Is a command that makes a copy of the current repository and places it in a separate location.
